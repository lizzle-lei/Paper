矩阵计算¶

核心的内容是 矩阵怎么求导数

导数

导数是切线的斜率

Tip

在机器学习当中，log默认的底数为e，即ln,其导数为1/x

亚导数

将导数扩展至不可微的函数

梯度

将导数扩展至向量

梯度的集合意义：与等高线正交（垂直），指向值变化最大的方向

向量与向量求导数时有两个重要的：

  1. y = A * x, y向量关于x的导数是A

  2. y = x的转置* A，y向量关于x的向量的导数是A的转置

Tip

凸函数可以拿到最优,不用去纠结最优解的问题

对于矩阵，向量的求导有两种布局

  1. 分子布局符号：分子y拉开不转置，分母x拉开要转置

  2. 分母布局符号：分母y拉开不转置，分子x拉开要转置

Tip

Pytorch和MXNet能够自动微分和计算图
